# Breathing app v1.0.0


## Live https://breathing-app-rails.herokuapp.com/
## Stack

- Ruby version 3.1.0
- Rails version 7.0.2.2
- CSS tailwind
- Jquery 



## Installation instructions :

```sh
git clone https://github.com/TomMard1337/Breathing-App.git     
cd Breathing-App
yarn add tailwindcss
bin/dev
```
